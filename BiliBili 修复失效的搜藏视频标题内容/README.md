这个脚本不会在B站找到被删除的视频的文件,只会让你知道搜藏的是什么的东西没了<br>

注:<br>

由于使用了ajax请求方式, 所以再打开别的搜藏栏的时候失效的视频依旧存在, 所以请点击那个 搜藏栏名字旁边的那蓝色文字修复吧!<br>
另外, 如果返回到原来的搜藏栏,修复的又失效了, 因为又Ajax了...<br>

至于什么是ajax. 那是和服务器相关的事儿了<br>
