工作原理<br>
根据 URL 上暴露出来的跳转链接，正则匹配提取真实的地址，例如知乎，Google<br>
如果 A 标签的内容为真实的地址，则替换，例如百度贴吧<br>
逐一发送请求，获取真实的地址，例如百度搜索<br>
根据请求特殊页面，这个特殊页面没有重定向地址，然后覆盖当前页，例如百度搜索，搜狗搜索<br>
覆盖原本的链接点击事件，比如 qq 邮箱<br>
