可以调整网页亮度，护眼，关闭浏览器再次打开可以还原上次的亮度。<br>
修改一个网页的亮度，其他网页都会改变亮度。<br>
按键：<br>
alt+↑(alt+方向键上) 提高亮度<br>
alt+↓(alt+方向键下) 降低亮度<br>
ctrl+alt+s 打开调整界面，鼠标点击亮度条调整亮度，点击空白地方关闭调整界面<br>
